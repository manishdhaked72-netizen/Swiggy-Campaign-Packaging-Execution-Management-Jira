# 🚀 Swiggy Campaign & Packaging Execution Management

## 📌 Overview
This project simulates a real-world brand marketing workflow inspired by Swiggy. It focuses on campaign execution, packaging coordination, agency management, and performance tracking using Agile methodology in Jira.

## 🛠 Tool Used
- Jira

## 🎯 Project Objectives
- Manage marketing campaign execution
- Coordinate packaging design with stakeholders
- Track agency deliverables and deadlines
- Plan and execute social media campaigns
- Monitor progress using dashboards

## 📂 Project Structure

### 🔥 Epics:
- Packaging Design Coordination
- Campaign Execution
- Agency Coordination
- Tracking & Reporting
- Social Media Calendar

### 📋 Key Activities:
- Created user stories and subtasks
- Assigned story points based on complexity
- Managed sprint planning and execution
- Tracked tasks using Agile board (To Do → In Progress → Done)
- Built dashboards (Burndown, Status Distribution, Workload)
